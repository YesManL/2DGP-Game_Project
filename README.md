# Sand Raider - 2D 탑뷰 액션 서바이벌 게임

## 프로젝트 개요
샌드 레이더는 끝없이 몰려오는 적들로부터 생존하는 2D 탑뷰 슈팅 게임입니다.

## 게임 조작법
- **WASD**: 플레이어 이동
- **마우스**: 조준 방향
- **마우스 좌클릭**: 총알 발사
- **ESC**: 게임 종료
- **SPACE**: 타이틀/게임오버 화면에서 시작/재시작

## 현재 구현된 기능 (1-3주차 프로토타입)

### 1주차: 기본 시스템
- ✅ 플레이어 캐릭터 이동 (WASD)
- ✅ 마우스 방향으로 회전
- ✅ 기본 배경 렌더링

### 2주차: 전투 시스템
- ✅ 마우스 클릭으로 총알 발사
- ✅ 적 캐릭터 생성 및 AI (플레이어 추적)
- ✅ 충돌 감지 시스템
- ✅ 적과 총알 간의 충돌 처리
- ✅ 플레이어와 적 간의 충돌 처리

### 3주차: UI 및 게임 흐름
- ✅ 타이틀 화면
- ✅ HP 표시 및 HP 바
- ✅ 적 처치 카운터
- ✅ 웨이브 시스템 기초
- ✅ 게임 오버 화면
- ✅ 화면 가장자리에서 적 스폰

## 게임 플레이
1. 타이틀 화면에서 SPACE를 눌러 게임 시작
2. WASD로 이동하며 마우스로 조준
3. 좌클릭으로 총알을 발사하여 적을 처치
4. 적이 플레이어에게 닿으면 HP가 감소
5. HP가 0이 되면 게임 오버
6. 게임 오버 화면에서 처치한 적의 수 확인
7. SPACE를 눌러 다시 시작

```
├── gameover_mode.py       # 게임 오버 화면
├── player.py              # 플레이어 클래스
├── enemy.py               # 적 클래스
├── bullet.py              # 총알 클래스
├── background.py          # 배경 클래스
└── ui.py                  # UI 클래스
```
